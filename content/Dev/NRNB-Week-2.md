Title: NRNB Academy Week-2
Date: 27-07-2016
Authors: Sourav Singh
Tags: NRNB,jsbml,Java

After a nice meeting with the mentors in the first week, We had our second meeting on Monday, 25th July. 
In the meeting, we discussed about the problem in setting up the Eclipse for jSBML and the mentor, Andreas Draeger, helped me in setting up Eclipse for the project.

Thereafter, I was given the task of writing programs in jSBML which can allow a user to display a SBML model and writing to a SBML file.

Writing the programs helped in understanding a lot about jSBML and its workings and also helped me in understanding the structure of the SBML file, which will come in 
handy when we need to create a PDF,PNG or LaTeX output of the SBML files.

Thereafter, it was decided that the work on SBML Reporter can be started. So I was assigned the task of Cloning the repository of SBML Reporter from Github and setting up Eclipse to run the project.

The NRNB Academy is going very nicely and I am getting to learn a lot of things and I am looking forward to learn a lot more.
