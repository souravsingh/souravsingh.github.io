After waiting for a long while for an available project and countless mails, I finally got to work on a project in NRNB Academy for the year 2016. 

The project that I am working on is-"SBML Reporter: a new systems biology model documentation tool" under the mentors Andreas Draeger and Michael Hucka.
More details on the project can be found over [here](https://github.com/nrnb/GoogleSummerOfCode/issues/23).

After discussions with the mentor, We decided to hold a Hangouts Video call on Monday,July 18. During the video call, 
We had a brief discussion about the project, what needs to be done for the project and some of the tasks required for the project.

After the discussion, I decide to spend the first week doing the following-

1. Reading the user guide for jSBML.
2. Setting up the Eclipse project for writing jSBML programs.
3. Reading about SBML2Latex and learning about its workings.

With this, I end the blog post and I am looking forward to learning more.
